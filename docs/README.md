# Let Us Python Book

This repository contains the source code and documentation for the "Let Us Python" book site.

## Documentation Site

The documentation is built using [MkDocs](https://www.mkdocs.org/) and deployed to GitHub Pages.

You can view the site here: [https://Joseph1997-eng.github.io/Let-Us-Python-Book-/](https://Joseph1997-eng.github.io/Let-Us-Python-Book-/)

## Local Development

To run the documentation locally:

1.  Install MkDocs:
    ```bash
    pip install mkdocs-material
    ```

2.  Serve the site:
    ```bash
    mkdocs serve
    ```

3.  Open your browser to `http://127.0.0.1:8000/`.

## Structure

- `mkdocs.yml`: Configuration file.
- `docs/`: Markdown source files.
- `docs/stylesheets/`: Custom CSS files.
